Welcome to Eclipse Client Docs

This is the official documentation for the Eclipse Client. Here you will find everything you need to know about installing, configuring, and using the mod. In case you need another version than 26.*, look at our changelog. You may find the right version but it may contain bugs or less features (or nuts).

Installation

To install the Eclipse Client, follow these simple steps:

  1. Download the latest version of fabric and of fabric api and our mod.
  2. Open your Minecraft mods folder.
  3. Drop the downloaded .jar file into the folder.
  4. Start the game!

Configuration

You can tweak the settings of the client by editing the config file located at config/eclipse.json or by tweaking it via commands:

{
  "fpscolor": 4095,
  "timercolor": 170,
  "fps": false,
  "sprint": true,
  "cacheSprint": false, // dont change this one
  "shown": false,
  "digital": true,
  "distance": 1.0,
  "ticks": 0,
  "seconds": 0,
  "minutes": 0,
  "hours": 0
}
